PROJETOS EM STUDIO MAX 3D

descrição do projeto

Esse é um projeto em 3d de um estande. Quando o cliente solicitava um estande para o vendedor era feito um brifing com a essencia que o cliente gostaria que tivese no projeto e a criação era por criatividade dos projetistas, no caso, eu.

Como podemos ver nesse caso o resultado ficou muito harmonico, sendo trabalhado estilos amadeirados, vidraças e aberturas, para que os clientes se sentissem convidados a estar no meio do estande e sentirem-se confortáveis.
